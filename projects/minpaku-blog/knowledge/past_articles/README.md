# 過去のベスト記事（Few-shot 例）

このフォルダに**過去の優れた記事**を `.md` 形式で保存すると、Claude がそれらを参考にして同じ品質・トーンで新規記事を生成します。

## 推奨

- ベスト3〜10記事を選んで保存
- 多すぎると逆効果（プロンプトが冗長化、トークン代増）
- 1ファイル＝1記事

## ファイル名

```
2024-12_minpaku-osaka-guide.md
2025-03_minpaku-tax-handling.md
```

YYYY-MM_テーマ.md の命名で並び順が分かりやすくなります。

## 何を入れる？

- ✅ 自分が書いて手応えのあった記事
- ✅ アクセスが伸びた記事
- ✅ ブランドボイスが綺麗に出ている記事
- ❌ 古くて法律情報が現状と違う記事（混乱の元）
- ❌ AI 生成された記事（ループになって品質劣化）

## 動作

`steps/claude.py` がこのフォルダの全 `.md` を読み込み、`generate_article` プロンプトに「過去記事の例」として埋め込みます。
